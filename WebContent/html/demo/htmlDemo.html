HTML <pre> 元素表示预格式化的文本，该文本将完全按照HTML文件中的格式显示。 通常使用非比例（“等宽字体”）字体呈现文本。 此元素内的空白显示为书面形式。

HTML <main>元素表示文档<body>的主要内容。 主要内容区域由与文档的中心主题或应用程序的中心功能直接相关或扩展的内容组成。

HTML <textarea>元素表示多行纯文本编辑控件，在您希望允许用户输入大量自由格式文本（例如，对评论或反馈表单的评论）时很有用。textarea的高度由rows来决定
